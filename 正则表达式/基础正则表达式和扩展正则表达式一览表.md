| 基础正则表达式   | 扩展正则表达式 | 含义          |
| --------- | ------- | ----------- |
| ^,$       | ^,$     | 以什么开始，以什么结束 |
| *         | *       | 大于等于0次      |
| &         | &       | 引用匹配的内容     |
| 无         | +       | 大于等于1次      |
| 无         | ？       | 大于等于0次      |
| 无         | \|      | 或者          |
| \ \{n,m\} | \{n,m}  | 匹配n到m次      |
| \ ( \ )   | ()      | 分组          |
|           |         |             |
